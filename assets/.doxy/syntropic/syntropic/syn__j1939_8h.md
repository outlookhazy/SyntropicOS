## Detailed Description


Implements J1939 29-bit CAN ID parsing/packing, PGN definitions, Transport Protocol (TP.CM BAM & RTS/CTS), Address Claiming (J1939-81), and Active Diagnostic Trouble Codes (DM1 J1939-73).
